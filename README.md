# 🔴 Pokémon Website — Assignment 1
A single-page Pokémon showcase built with pure HTML & CSS, featuring a split-color hero section with Pikachu, a custom Pokémon-style logo, and a decorative pokéball SVG watermark.
## 🌐 Live Preview
Open `index.html` directly in your browser to view the site.
## 🛠️ Built With
- **HTML5**
- **CSS3** — Flexbox/Grid layout, `-webkit-text-stroke` for the logo, inline SVG for the pokéball icon
- **Google Fonts** — [Baloo 2](https://fonts.google.com/specimen/Baloo+2)
## 📂 Files
Assignment-1-Pokemon-website-/
├── index.html # Page structure & content
├── style.css # Styling (layout, colors, logo, hero, pokéball watermark)
└── Pokemon-Pikachu-PNG.png # Pikachu artwork used in the hero section
## ✨ Features
- Two-tone split background (deep yellow / pale yellow)
- Custom "Pokémon" wordmark logo built entirely in CSS (no logo image)
- Pikachu artwork breaking across the color divide
- Pagination indicator (`01 / 08`) with dot navigation
- Faint pokéball SVG watermark in the bottom-left corner
- Hamburger menu icon (top right)
## 🚀 Getting Started
Clone the repo and open it locally:
```bash
git clone https://github.com/dikshaasnora/Assignment-1-Pokemon-website-.git
cd Assignment-1-Pokemon-website-
```
Then just open `index.html` in your browser — no build step or dependencies required.
